template<typename... Args>
GW::GReturn GW::GRAPHICS::GRasterSurface::UpdateSurface ( Args &&...  parameter)

Draws a raster to the screen.

BLITs a pixel data array from the raster to internal memory, then draws it to the screen as soon as the window is ready. Note: On Linux, in order to take advantage of the performance increase from double buffering, call this function on another thread. This is not necessary on other platforms.

Parameters
[in]_argbPixelsThe pixel data array to draw to the screen.
[in]_numPixelsThe number of pixels in the array.
Return values
GReturn::SUCCESSPixel data was passed to the surface.
GReturn::INVALID_ARGUMENTInvalid pixel data and/or pixel count higher than window pixel count was passed.
GReturn::PREMATURE_DEALLOCATIONThe associated GWindow was deallocated and the surface is now invalid.
GReturn::UNEXPECTED_RESULTThe function entered an area it was not supposed to be able to.

◆ SmartUpdateSurface()

template<typename... Args>
GW::GReturn GW::GRAPHICS::GRasterSurface::SmartUpdateSurface ( Args &&...  parameter)

Determines how to update from bit flags, BLITs pixel data to internal memory, and draws it to screen.

BLITs a pixel data array from a raster to internal memory, processes it as specified by bit flags, then draws it to the screen as soon as the window is ready. Note: On Linux, in order to take advantage of the performance increase from double buffering, call this function on another thread. This is not necessary on other platforms.

Parameters
[in]_argbPixelsThe pixel data array to draw to the screen.
[in]_numPixelsThe number of pixels in the data array to draw to the screen.
[in]_rowWidthThe width of each row in the pixel data array to draw to the screen.
[in]_presentOptionFlagsA bit flag set specifying how to process the pixel data before drawing.
Return values
GReturn::SUCCESSPixel data was passed to the surface.
GReturn::INVALID_ARGUMENTInvalid pixel data, pixel count higher than window pixel count, width larger than the window, and/or conflicting bit flags were passed to the function.
GReturn::PREMATURE_DEALLOCATIONThe associated GWindow was deallocated and the surface is now invalid.
GReturn::UNEXPECTED_RESULTThe function entered an area it was not supposed to be able to.
