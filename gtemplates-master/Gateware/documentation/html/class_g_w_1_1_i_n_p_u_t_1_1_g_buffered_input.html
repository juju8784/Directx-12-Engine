typedef forceRaw:: Events GW::INPUT::GBufferedInput::Events

Events generated by GBufferedInput.

These messages are used to notify observers when tasks complete. 

Events::KEYPRESSED          A key pressed event.
Events::KEYRELEASED         A key released event.
Events::BUTTONPRESSED       A Mouse/Pointer Button pressed event.
Events::BUTTONRELEASED      A Mouse/Pointer Button released event.
Events::MOUSEMOVE           A Mouse/Pointer move event.
Events::MOUSESCROLL         A Mouse/Pointer scroll event.

typedef forceRaw:: EVENT_DATA GW::INPUT::GBufferedInput::EVENT_DATA

EVENT_DATA provided by GBufferedInput.

All events provide the following relevant information. 

EVENT_DATA::data        Data storing the key/button information.
EVENT_DATA::x           Window Mouse position x when event is sent.
EVENT_DATA::y           Window Mouse position y when event is sent.
EVENT_DATA::screenX     Screen Mouse position x when event is sent.
EVENT_DATA::screenY     Screen Mouse position y when event is sent.
EVENT_DATA::keyMask     Bit flags for (CapsLock, NumLock, ScrollLock, Shift, and Control).

See GInputDefines.h for list of keyMask defines.
