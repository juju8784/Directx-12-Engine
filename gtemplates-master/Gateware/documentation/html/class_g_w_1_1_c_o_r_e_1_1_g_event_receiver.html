Detailed Description

Receives any & all events produced by an interface derived from GEventGenerator. Accetps an std::function<void()> as the callback function/lambda for flexibility. Use std::bind to supply your own arguments to your callback, the only restriction is the void return type. To receive more detailed information, embbed this Proxy in your callback and "Pop()" the most recent event.

MyEventReceiver.Create( MyEventGenerator, [](MyEventReceiver&) {
GEvent g;
GEventGenerator::Events e; // substitute the generator type you care about
GEventGenerator::EVENT_DATA d;
if(+MyEventReceiver.Pop(g))
if(+g.Read(e,d))
if(e == GEventGenerator::Events::NON_EVENT) // substiute actual event
std::cout << d.noData; // substitute actual data
});

Allocates & Intializes a GEventReceiver.

A GEventReciever is capable of receiving and reacting to events created by a GEventGenerator. You can use this class whenever there is a Gateware class that spawns events you wish to know about.

Parameters

Return values
